## 📱 Applications Included

### 1. GeoQuiz
An interactive geography quiz application.

Features:
- Multiple choice geography questions
- Score tracking
- Multi-screen navigation
- Simple and intuitive UI

Technologies:
- Kotlin
- Android Studio
- Android Activity lifecycle

---

### 2. GeoQuizPractice
Practice version of the GeoQuiz app used to test new features and UI improvements.

Features:
- Question navigation
- Practice mode for quiz testing
- UI experimentation

---

### 3. Criminal Intent App
An Android application for recording and managing crime reports.

Features:
- Create and edit crime entries
- Store reports using local database
- Photo capture functionality
- Location tagging

Technologies:
- Kotlin
- SQLite Database
- RecyclerView
- Camera API

---

### 4. Photo Gallery App
Displays photos fetched from an online API.

Features:
- Fetch images from public API
- Display 100+ images
- RecyclerView grid layout
- Asynchronous loading

Technologies:
- Kotlin
- REST API
- RecyclerView
- Async Tasks / Coroutines

---

### 5. Animation App
An Android application demonstrating UI animations and transitions.

Features:
- Smooth UI transitions
- Animated components
- Interactive UI states

Technologies:
- Kotlin
- Android Animation Framework

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/IntekhabIA/android-kotlin-projects.git
```

2. Open the project in **Android Studio**

3. Sync Gradle and run the desired app on an emulator or Android device.
